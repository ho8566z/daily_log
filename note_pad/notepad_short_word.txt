+단축키 모음

-ctrl + b : explore로 이동하기

-ctrl + F5 : MS VScode애서 'Run play file'실행하기

-ctrl + / : 원하는 문자, 코드를 드래그하고, 적용하면, 전부 '#'처리되어 '주석'화됨

-shift + alt + 방향키 : 원하는 문자, 코드를 드래그하고, 원하는 방향키를 누르면, 
원하는 만큼, 복사됨

-ctrl + z : 이전의 상태로 돌아가기(ex:'이전 페이지로 이동하기'와 같음)

-ctrl + / : 원하는 부분을 드래그하고 실행하면 전체 주석상태로 변경/재변경 가능

-ctrl + a : 전체 선택

-ctrl + shift + 방향키 : 한번에 키보드 입력, 이동 가능

-alt + 방향키 : 선택한 '행'을 전체 이동 가능

-# ai 질문할 것

-ctrl + ` : MS VScode의 터미널 단축키
