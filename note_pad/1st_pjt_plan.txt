	-등록한 사람의 얼굴인식
	-지명수배자 탐색
	-실종자 탐색
	-치매 환자 탐색

	-이거 불가능하면 -> 매사냥

[벤치마킹]
1. 황금방패: 원치 않는 사람까지 감지함
	-우리는 안그럼 (보안대책)

2. 신원 인식: 기존 방식(직원카드, 지문인식 등)
	-> 실시간 탐지는 불가능

3. 페이스 인증
	-> 포즈, 인상, 해상도 등이 변하거나 거리감이 변하거나 하면 인식 실패

4. 회원가입?(아이디, 패스워드)

해당 기술은 컴퓨터 비전(Computer Vision)의 핵심 분야들로 구성되어 있으며, 
구체적인 핵심 기술과 구현 프로세스는 다음과 같다.

[핵심 기술 요소]
	-객체 탐지 (Object Detection): 영상 속에서 '선수(Player)'와 '공(Ball)'의 
		위치를 실시간으로 찾아내고 사각형 박스(Bounding Box)를 치는 기술이다. 
			대표적으로 YOLO(You Only Look Once) 시리즈가 가장 많이 사용된다.
	-객체 추적 (Multi-Object Tracking, MOT): 단순히 찾는 것을 넘어, 프레임이 바뀌어도 
		'1번 선수', '2번 선수'처럼 고유한 ID를 부여해 지속적으로 추적하는 기술이다. 
			영상 내의 ByteTrack이나 DeepSORT 같은 알고리즘이 여기에 해당한다.
	-모델 최적화 및 양자화 (Quantization): 실시간으로 수많은 선수와 빠르게 움직이는 공을 
		끊김 없이 감지하기 위해, AI 모델의 용량을 줄이고 연산 속도를 극대화하는 기술이다. 
			(예: FP32 데이터를 INT8로 변환하여 NPU나 Edge 디바이스에서 고속 추론)
									
[구현 난이도 및 접근 방법]
오픈소스 생태계가 매우 잘 구축되어 있어, 개발 경험이 있다면 비교적 빠르게 프로토타입을 
	만들어볼 수 있다.
		:bulb: 단계별 구현 로직

	-데이터 수집 및 라벨링: 축구 경기 영상에서 선수와 공 영역에 바운딩 박스를 치고 학습 
		데이터를 준비한다. 
			(이미 잘 만들어진 COCO 데이터셋이나 축구 전용 데이터셋을 활용할 수도 있다.)
	-모델 학습: YOLOv8 또는 YOLOv11 같은 최신 객체 탐지 모델에 준비한 데이터를 학습시킨다.
	-트래커 결합: 학습된 YOLO 모델의 출력값을 ByteTrack 소스 코드와 연동하여, 탐지된 
		객체에 ID를 부여하고 추적하게 만든다.
			:bulb: 한 걸음 더:

실시간 경기 분석 서비스나 모바일 환경에서 부드럽게 구동하려면, TensorRT나 ONNX 변환 및 
	모델 양자화 과정을 거쳐 하드웨어(GPU/NPU) 연산 효율을 극대화하는 과정이 추가로 필요하다.

영상 속 기술은 보통 다음 조합과 같다

[------------------------------]
카메라 영상
 ↓
사람 탐지 (YOLO)
 ↓
객체 추적 (DeepSORT, ByteTrack)
 ↓
ID 부여
 ↓
시각화예시:
선수 A → ID 8
선수 B → ID 12
선수 C → ID 25

위 선수들이 움직여도 같은 ID를 유지한다.
[------------------------------]

Hikvision: 중국의 CCTV, IP 카메라, NVR, DVR, 출입통제 시스템 등을 제조하는 세계적인 
	영상보안 기업
	
	-AcuSens: AI 영상분석 기술(카메라 또는 NVR 내부에 딥러닝 AI를 탑재하여 다음을 수행하는 기술)
		-사람 검출
		-차량 검출
		-침입 감지: 침입 발생 시 조명이나 사이렌을 통해 사전에 경고할 수 있으며, 사람이나 
			차량을 기준으로 빠르게 과거 녹화 영상을 검색할 수 있습니다.
		-라인 크로싱(Line Crossing)
		-사람/차량 분류: 사람과 차량의 움직임을 정밀하게 식별하여, 그 외의 물체
			(동물, 비, 나뭇잎 등)로 인한 불필요한 알람을 필터링합니다
		-오탐 감소(나뭇잎, 비, 그림자, 동물 등): 보안 인력이 실제 발생한 위협에만 
			집중할 수 있도록 도와주며, 관제 효율을 크게 높여줍니다
			
			

	-참고: https://www.hikvision.com/korean/core-technologies/see-smarter-technology/acusense/

			특징이나 차량의 차종(트럭, 승용차, 자전거 등) 및 색상으로 대상을 찾을 수

			해당 대상과 일치하는 동영상 내 위치를 역추적합니다.
		-타임라인 및 동선 추적: 여러 대의 감시 카메라에 포착된 대상의 이동 경로를
